# Release

## AnkiWeb Upload

### Prerequisites

- AnkiWeb account
- Screenshots for the listing

### Steps

1. **Build the addon package**

   ```bash
   pnpm build-addon
   ```

2. **Upload to AnkiWeb**
   - Go to https://ankiweb.net/shared/upload
   - Upload `anki-browse-web.ankiaddon`
   - Fill in the listing details:
     - Title: `Browse Web`
     - Description (see below)
     - Screenshots

3. **Update README.md**
   - Add screenshots to Demo section
   - Add link to AnkiWeb listing
   - Remove the TODO about sharing
